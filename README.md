# About 

With this project, I implemented real-time procedural FX motion graphics in UE using blueprints.

# Concepts Learned
1. Motion VFX
2. Proceduralism
3. Material Graphs 
4. Cube Split
5. Dynamic Material
6. Optimisation and NVIDIA DLSS Plugin
7. UI and Widgets
